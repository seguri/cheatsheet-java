Java Cheat Sheet

Strings

Sort characters in a string

var chars = aString.toCharArray();
Arrays.sort(chars);
var sortedString = new String(chars);

Random

All examples use aRandom = new Random().

Random from half-closed interval [a, b)

min + aRandom.nextInt(max - min);

Random from closed interval [a, b]

min + aRandom.nextInt(max - min + 1);

Random element from a list

aList.get(aRandom.nextInt(aList.size()));

Random element from a collection

aCollection.stream()
  .skip(aRandom.nextInt(aCollection.size()))
  .findFirst();

Maths

Involution

A function is involutory if, applied twice, returns the original value: f(f(x)) = x.

Examples:

  • Identity: f(x) = x
  • Negation: f(x) = -x
  • Binary search insertion point: f(x) = -(x + 1)
